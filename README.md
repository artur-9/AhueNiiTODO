# 📝 TODO List (Laravel + Docker)

## 🚀 Быстрый запуск:

1. `cp .env.example .env`
2. `docker compose up -d --build`
3. Подожди 30 секунд
4. Открой http://localhost:8000

## 🔧 Если не работает:
- Очисти кэш браузера (Ctrl+Shift+Delete)
- Или открой в режиме инкогнито

## 🗄️ База данных:
- Хост: `db`
- Порт: `3307` (для внешнего доступа)
- Данные загружаются автоматически из `docker/mysql/init/`
